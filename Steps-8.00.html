  • Replace null values.
Detailed instructions:

1.       In Power BI Desktop Home ribbon, click on Transform data. This will open Power Query.
2.       Select the columns with the county names. The OBJECTID and DATE columns do not have null values.
3.       Click on Replace Values on the Transform ribbon.
4.       In the Replace Values window, type null in the Value to Find space, and 0 in the Replace With space. Finish by clicking OK.

  • Removing unused columns
This purpose of this dashboard is to help visualize new Covid-19 deaths in all 24 counties of Maryland. Therefore, unused columns are removed. These are the columns with the cumulative total of deaths in each county. We are left with daily new deaths columns. The Unknown column, which is the last column in the table, is also removed in this step.

Detailed instructions:
  1. In Power Query, select the columns with Cumulative deaths. These are the columns that have Cumulative in the heading.
  2. In the Home ribbon, click on Remove columns and choose Remove columns. 
  3. Click on Close & Apply.

  • Creating Year, Quarter and Month columns.


Detailed instructions:


1.       In Power BI Desktop, click on Transform data.
2.       In Power Query, choose the table and go to Add Column and double click on Column From Example.
3.       A new column appears in the table. Rename it State and type Maryland in the 1st two rows of the column. Then the value will automatically show on the remaining rows. Click on OK and the rest of the column will be filled. Use the same steps to create the Sate Code (MD) column. Then click on Close & Apply to save.
4.       In Power BI Desktop, click on New column in the Home ribbon.
5.       Type the following code to create a calculated column. Name it County and State.
 
County and State = 'MD_COVID19_DEATHS'[County] & "," & " " & 'MD_COVID19_DEATHS'[State Code]
 
6.       When done, click on Save.
