Create Calendar table 10



Detailed Instructions:
1.       In Power BI Desktop, click on Modeling ribbon and select New table.
2.       In the formula bar, enter the following code and enter to save:
 
Calendar = ADDCOLUMNS (
CALENDAR (Min('MD_COVID19_DEATHS'[DATE]), MAX('MD_COVID19_DEATHS'[DATE])),
"Year", YEAR ( [Date] ),
"Monthnumber", FORMAT ( [Date], "MM" ),
"YearMonthShort", FORMAT ( [Date], "YYYY/mmm" ),
"MonthNameShort", FORMAT ( [Date], "mmm" ),
"Quarter", "Q" & FORMAT ( [Date], "Q" ),
"YearQuarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" )
)
