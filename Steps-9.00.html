Create Refresh Date Table 9



Detailed Instructions:
1.       In Power BI Desktop Home ribbon, click on Get data and select Blank Query. Power Query will open.
2.       Under Properties, replace Query 1 by Last Refresh Date in the Name box.
3.       Next, go to the formula bar and type the following DAX code and click enter: 
=DateTime.LocalNow()
4.       The current date and time will be displayed. Click on Convert To Table under the DateTime Tools Tab. 
5.       Rename Column1 as Last Refresh Date Time and change the data type to Date/Time. Click Close & Apply to save.
