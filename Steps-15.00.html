Add Project Title, Project Logo and Colaberry Logo 15


Detailed Instructions:

1.       To add the project title, go to Power BI Desktop Home ribbon and click on Text box. 
2.       Type the title in all caps: MD COVID-19 CONFIRMED DEATHS (2020 – 2023).
3.       In the Format pane, go to the General tab. 
  • In Properties: Size: Height – 63; Width – 1280; Position: Horizontal – 0; Vertical – 0; Title: Off.
  • Font: Segoe UI; Font size: 20; Font color: #666666; Center aligned.
4.       To add project logo and Colaberry logo, click on the Insert ribbon in Power BI Desktop.
5.       Click on Image. Choose MD Covid logo and Colaberry logo from the Project Essentials folder.
6.       For MD Covid logo
  • In Format pane, in Image tab, click on Style. In Scaling, select Fit.
  • In General tab, Properties: Size: Height – 62; Width – 194; Position: Horizontal – 0; Vertical – 1; Title: Off.
7.       For Colaberry logo
  • In Format pane, General tab, Properties: Size: Height – 62; Width – 194; Position: Horizontal – 1074; Vertical – 1; Title: Off.
