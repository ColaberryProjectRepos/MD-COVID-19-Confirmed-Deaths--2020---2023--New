Create Measures 13

Detailed Instructions:

1.       In Power BI Desktop, click on the MD_COVID19Deaths table in the Data Pane.
2.       In the Home ribbon, click on New measure. 
3.       Type the following DAX codes in the formula bar one at a time and enter to save: 

  • Total Deaths = CALCULATE(SUM('MD_COVID19_DEATHS'[Deaths]))
  •  Death Rate = DIVIDE(SUM('MD_COVID19_DEATHS'[Deaths]), SUM(‘Stats’ [Covid Cases]), 0) * 100

  • Total Deaths YoY% = 
           VAR __PREV_YEAR = CALCULATE(SUM('MD_COVID19_DEATHS'[Deaths]),          DATEADD('Calendar'[Date], -1, YEAR))
            RETURN
                DIVIDE(SUM('MD_COVID19_DEATHS'[Deaths]) - __PREV_YEAR, __PREV_YEAR)
  • Total Death QoQ% = 
       VAR __PREV_QUARTER =
              CALCULATE([Total Deaths],DATEADD('Calendar'[Date], -1, QUARTER)
               )
       RETURN
              DIVIDE([Total Deaths] - __PREV_QUARTER, __PREV_QUARTER)

4.       To create an arrow for Total Death YoY% trend, type the following code in the formula bar to create a measure:

                                        


Arrow YOY% = SWITCH(TRUE(), [Total Deaths YoY%] <0, UNICHAR(9662),[Total Deaths YoY%] =0, UNICHAR(9666), UNICHAR(9652))
